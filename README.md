# Kevin's Engineering Command Center

![Status](https://img.shields.io/badge/Status-Active-brightgreen?style=flat-square)
![HTML5](https://img.shields.io/badge/HTML5-Redirect%20Page-E34F26?style=flat-square&logo=html5)

A redirect and splash page for **Kevin's Engineering Command Center** — an engineering dashboard and productivity tool. This repository hosts the GitHub Pages entry point that routes users to the live application on Replit.

---

## Features

- Animated spinner loading screen with 3-second delay
- Automatic redirect to the live app
- Dark blue gradient theme (`#0f2027` → `#203a43` → `#2c5364`)
- Single-file static HTML — zero dependencies

## Live App

Visit the live application: [kevinsengineeringcommandcenter.replit.app](https://kevinsengineeringcommandcenter.replit.app/)

## How to Run Locally

```bash
git clone https://github.com/danielmiranda24b/KevinsEngineeringComandCenter.git
cd KevinsEngineeringComandCenter
# Open index.html in your browser
```
